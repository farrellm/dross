# Dross
Integrated note taking and document archive using org-node. LLM augmented Zettelkasten Method.

## Backend
- data stored as emacs org-node notes
- interface exposed as an MCP server (Haskell)
- agent client protocol

## Frontend
- user initiated interactions via Claude Code, specialized CLAUDE.md
- agent initiated via a Telegram bot (Go)
- maybe also a React webapp for ?
